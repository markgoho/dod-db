{{- /* Build a CSS-ident-safe view-transition-name. `urlize` alone is not enough: it preserves characters like `.` that are valid in URL slugs but invalid in CSS custom idents, which causes browsers to silently drop the declaration and skip the transition. We coerce any non-ident character to a hyphen so the same input always produces the same safe name on both ends of a navigation. Args: prefix – required, e.g. "guests-title" source – required, raw value to slugify (usually .Title or an ID) */ -}} {{- $prefix := .prefix -}} {{- $source := .source -}} {{- $raw := printf "%s-%s" $prefix (printf "%v" $source | urlize) -}} {{- return (replaceRE `[^a-zA-Z0-9_-]` "-" $raw | safeCSS) -}}
